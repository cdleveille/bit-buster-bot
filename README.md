# bit-buster-bot

## Setup

1. Install [bun](https://bun.sh)
2. Install package dependencies:
   ```bash
   bun install
   ```
3. Create a `.env` file in the root directory and add your Discord bot token:
   ```
   BOT_TOKEN=*YOUR_DISCORD_BOT_TOKEN*
   ```
4. Start the bot:
   ```bash
   bun dev
   ```
